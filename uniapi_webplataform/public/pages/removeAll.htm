Borrar todos los datos

Si deseas pulsando el boton borrar. Puedes borrar todos los datos de la base de datos, todos sus datos seran eliminados y será dispuesto para luego una cuenta administradora llamada 'ADMIN', con contraseña 'ADMIN', misma que el nombre de usuario. Es aconsejable despues del borrado, se modifique la contraseña a una mas robusta.

Pregunta de seguridad
